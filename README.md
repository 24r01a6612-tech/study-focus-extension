# 🎯 Study Focus Chrome Extension

A productivity Chrome extension that helps users study using:

- ⏱️ Pomodoro Timer (25 min)
- 🔥 Streak Tracking System
- 🏆 Badge Rewards System
- 📊 Study Analytics Graph
- 🔊 Focus & Distraction Sounds
- 🚨 YouTube/Instagram Focus Detection

## 🚀 Features

- Gamified study system
- Burnout protection alerts
- Motivational feedback system
- Chrome toolbar badge icon updates

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Chrome Extensions API
